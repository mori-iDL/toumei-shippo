# とうめいしっぽ — パラメトリックパネルデザイナー

## プロジェクト概要

**事業名**: 盛岡市森林づかいイノベーション事業（補助金事業 R6〜R8）
**プロダクト名**: とうめいしっぽ
**ワンフレーズ**: 触れて、選んで、育てる——コードが生む形を、森の木が受けとめる

赤ちゃんの誕生データ（名前・誕生日・体重・身長・出生地・天気）を、自然計算アルゴリズムで木の模様に翻訳するシステム。「おもちゃ」ではなく「おもちゃが生まれ続けるシステム」を作る。

---

## 設計哲学: Digital Nature

すべてのデザイン判断は以下の哲学に基づく:

- **有機的**: 完全な対称・規則性を避ける。自然の揺らぎを内包する
- **非対称**: 左右対称に見えて微妙にずれている。木目のように
- **自然の不規則性**: ノイズ、乱数、カオスは味方。ただし制御された範囲で
- **時間が育てる価値**: Timeless / Longline design。10年後も美しいこと
- **素材の本質**: 木は「懐の深い入れ物」。デジタルパターンを受け止める器

**判断に迷ったとき**: 「10年後のZineが誇れるデザインか？」

## 三層翻訳アーキテクチャ

赤ちゃんの物語を木の模様に変換する三層構造。各層は独立しつつ相互に影響する。

### Layer 1: 赤ちゃんの伝記的声（入力パラメータ）

| パラメータ | 変換先 | 詳細 |
|-----------|--------|------|
| 名前（ひらがな） | 母音分析 → 副アルゴリズム選択 | あ=フローフィールド, い=正弦波, う=パーリンノイズ, え=リサージュ, お=波動関数 |
| 誕生日 | 季節判定 → 主アルゴリズム選択 | 春=フィボナッチ螺旋, 夏=ボロノイ, 秋=反応拡散, 冬=フラクタル分岐 |
| 出生時刻 | ブレンド比率 | 正午/真夜中=純粋な主アルゴリズム, 夜明け/夕暮れ=最大混合 |
| 体重 | 力強さ / 繊細さ | 軽い(2000g)→細い線・小さな要素・繊細な反復、重い(5000g)→太い線・大きな要素・力強い存在感。基準: 3000g=中立 |
| 身長 | 伸びやかさ / 凝縮 | 短い(40cm)→中心に凝縮・密な要素、長い(60cm)→外に展開・長いストローク・開放感。基準: 49cm=中立 |
| 出生地 | 都道府県 | 将来的に地域性反映（現在は表示のみ） |
| 天気 | 乱流度 | 晴れ=穏やか, 雨=有機的揺らぎ, 雪=結晶的, 曇り=柔らかい拡散 |

### Layer 2: 自然計算アルゴリズム（変換エンジン）

**主アルゴリズム**（季節で決定）:
- **春 — フィボナッチ螺旋**: 成長のリズム、黄金比の展開
- **夏 — ボロノイ**: 細胞分裂、領域の主張、力強い区画
- **秋 — 反応拡散**: チューリングパターン、有機的な斑模様
- **冬 — フラクタル分岐**: 枯木の枝、再帰的な分岐構造

**副アルゴリズム**（名前の母音で決定）:
- **あ — フローフィールド**: 風のように流れる場
- **い — 正弦波**: 規則的な振動、リズム
- **う — パーリンノイズ**: 自然な揺らぎ、地形的
- **え — リサージュ**: 交差する軌跡、幾何学的な美
- **お — 波動関数**: 波紋のような同心円的広がり

**母音ブレンド**: 名前の全母音を重み付きで合成（先頭母音=主旋律、後続母音=和音）。出生時刻で主副の混合比を決定。
**体重・身長**: 量的変化（点の数・半径）ではなく、質的変化で影響。体重→線の太さ・要素サイズ・構造の粒度。身長→構図の凝縮/展開・ストロークの長さ・パターンの異方性。

### Layer 3: 木の固有特性（出力メディア）

9つの岩手県産樹種。各樹種にlineSpacing / lineWeight / grainStrength設定あり。

| 樹種 | 特性 | 加工適性 |
|------|------|---------|
| 杉（スギ） | 柔らかい木目、温かい色調 | レーザー◎ V溝NC◎ |
| 栗（クリ） | 力強い年輪、タンニン豊富 | レーザー○ V溝NC◎ |
| 欅（ケヤキ） | 美しい杢目、堅牢 | レーザー△ V溝NC◎ |
| 桐（キリ） | 極めて軽い、白い | レーザー◎ V溝NC○ |
| 胡桃（クルミ） | 深い褐色、滑らか | レーザー○ V溝NC◎ |
| 楢（ナラ） | 虎斑杢、堅い | レーザー△ V溝NC◎ |
| 桜（サクラ） | 赤みのある暖色、緻密 | レーザー○ V溝NC◎ |
| 栃（トチ） | 絹のような光沢、縮み杢 | レーザー○ V溝NC◎ |
| 唐松（カラマツ） | 脂分、赤褐色、強靭 | レーザー○ V溝NC◎ |

## 二重出力モード

### レーザーモード
- 精密ベクトルライン、均一線幅
- 背景: #f5ede0（明るい木肌色）
- 出力形式: SVG, PNG
- 用途: 繊細な模様、量産向き

### V溝NCモード
- 可変幅・可変深さライン、立体的表現
- 背景: #e8dcc8（やや濃い木肌色）
- 出力形式: SVG, DXF
- 用途: 彫刻的表現、高級感、一品物

### セッション開始時の行動（Phase A: 文脈取得）

SessionStartフックの指示に従い、以下を**能動的に**取得する：

1. **プロジェクトHub**をfetchし、最新の意思決定ログとロードマップを確認
2. **タスクDB**で `担当=松森` × `ステータス=未着手 or 進行中` を検索し、作業候補を把握

1. タスクのステータスを更新（未着手→進行中→完了）
2. タスクにコメント追加（作業内容の要約 + commitハッシュ）
3. 年輪DB記録（既存フロー）
4. Handoff更新（既存フロー）

## コーディング規約

### HTML/CSS/JS
- シングルファイル構成を維持（ポータビリティ最優先）
- CSS変数でテーマ管理（:root内に集約）
- Canvas描画関数は純粋関数に近づける（同じ入力→同じ出力）
- イベントは`change`で統一（`<select>`は`input`が発火しない）
- デバウンス50msを維持

### アルゴリズム実装
- 各アルゴリズムは独立した関数として実装
- シード値は入力パラメータから決定論的に生成（Math.random()禁止）
- パラメータ範囲は明示的に定数で定義（マジックナンバー禁止）
- `fn.length`でアルゴリズム関数のシグネチャを動的検出するパターンを使用
- `normalizeWeight(w)` / `normalizeHeight(h)` で -1〜+1 に正規化。中心値は日本人新生児平均（3000g/49cm）

### 命名規則
- アルゴリズム名: 日本語コメント + 英語関数名
- 樹種データ: `WOOD_SPECIES`定数オブジェクト
- パラメータ: `PARAM_RANGES`定数で最小・最大・デフォルトを定義
