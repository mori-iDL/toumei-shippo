# 岩手県産9樹種データベース

## 樹種パラメータ一覧

以下はHTMLデザイナーの`WOOD_SPECIES`定数に対応するデータ。

### 杉（スギ / Cryptomeria japonica）
- **木目特性**: 直線的、均一な年輪間隔、柔らかい印象
- **色調**: 白太〜赤身、温かみのある色調
- **硬度**: 軟（加工容易）
- **lineSpacing**: 3.0（広め＝柔らかい木目表現）
- **lineWeight**: 0.8（細め）
- **grainStrength**: 0.3（控えめな木目干渉）
- **レーザー加工**: ◎ 焦げが美しい、均一に彫れる
- **V溝NC加工**: ◎ 柔らかく切削しやすい
- **文化的意味**: 日本の代表的建築材、素直さ、まっすぐさ

### 栗（クリ / Castanea crenata）
- **木目特性**: 力強い環孔、明瞭な年輪、ダイナミック
- **色調**: 黄褐色、タンニンで経年変化が美しい
- **硬度**: 中〜硬
- **lineSpacing**: 2.5
- **lineWeight**: 1.2（太め＝力強い年輪を表現）
- **grainStrength**: 0.6（木目が模様に影響する）
- **レーザー加工**: ○ タンニンで焦げやすいが味がある
- **V溝NC加工**: ◎ 粘りがあり美しい切削面
- **文化的意味**: 縄文時代からの食と住の木、生命力

### 欅（ケヤキ / Zelkova serrata）
- **木目特性**: 美しい杢目（玉杢・如鱗杢）、複雑で豊か
- **色調**: 赤褐色〜黄褐色
- **硬度**: 硬
- **lineSpacing**: 2.0（密め＝杢目の精緻さ）
- **lineWeight**: 1.0
- **grainStrength**: 0.8（木目が強く模様と干渉）
- **レーザー加工**: △ 硬くて深い彫りが難しい
- **V溝NC加工**: ◎ 硬いが美しい光沢の切削面
- **文化的意味**: 寺社仏閣の材、格式と威厳

### 赤松（アカマツ / Pinus densiflora）
- **木目特性**: 明瞭な年輪、脂条がある、温かみのある木目
- **色調**: 黄白色〜淡い赤褐色、脂分で飴色に変化
- **硬度**: 中（針葉樹としては硬め）
- **分類**: 針葉樹
- **lineSpacing**: 2.8（中程度）
- **lineWeight**: 0.9
- **grainStrength**: 0.45（脂条が木目に表情を与える）
- **レーザー加工**: ○ 脂分で焦げにムラが出ることも（味になる）
- **V溝NC加工**: ◎ 適度な硬さで切削しやすい
- **文化的意味**: 岩手の里山の象徴、松林の風景、粘り強さ

### 鬼胡桃（オニグルミ / Juglans ailantifolia）
- **木目特性**: 滑らかで落ち着いた木目、上品な散孔材
- **色調**: 灰褐色〜深い褐色、オイル仕上げでチョコレート色に深化
- **硬度**: 中（気乾密度 0.53g/cm³）
- **分類**: 散孔材・広葉樹
- **lineSpacing**: 2.8
- **lineWeight**: 0.9
- **grainStrength**: 0.4（穏やかで均一な木目）
- **レーザー加工**: ○ 均一に彫れる、焦げ色が深く味わいがある
- **V溝NC加工**: ◎ 滑らかで美しい切削面、粘りが適度
- **経年変化**: 灰褐色→飴色に穏やかに変化、オイルで深みが増す
- **文化的意味**: 食べ物としても親しまれる岩手の里山の恵み、縄文期からの利用

### 楢（ナラ / Quercus crispula）
- **木目特性**: 虎斑杢が特徴的、力強い放射組織
- **色調**: 黄褐色
- **硬度**: 硬
- **lineSpacing**: 2.2
- **lineWeight**: 1.1
- **grainStrength**: 0.7（虎斑が模様に強く干渉）
- **レーザー加工**: △ 硬い、深彫りに不向き
- **V溝NC加工**: ◎ 虎斑と彫刻の組み合わせが美しい
- **文化的意味**: ドングリの木、子供の遊び、森の象徴

### 山桜（ヤマザクラ / Cerasus jamasakura）
- **木目特性**: 緻密で均一、穏やかな散孔材の木目
- **色調**: 赤みのある暖色、ピンクがかった褐色
- **硬度**: 中〜硬（気乾密度 0.60g/cm³）
- **分類**: 散孔材・広葉樹
- **lineSpacing**: 2.4
- **lineWeight**: 0.9
- **grainStrength**: 0.35（穏やかで繊細な木目）
- **レーザー加工**: ○ 均一で美しい焦げ色、焼きムラが少ない
- **V溝NC加工**: ◎ 緻密で繊細な表現が可能、光沢のある切削面
- **経年変化**: ピンク褐色→深い赤褐色へ、紫外線で色が深まる
- **文化的意味**: 日本の象徴、出会いと別れ、新しい始まり

### 板屋楓（イタヤカエデ / Acer pictum）
- **木目特性**: 緻密で均一、絹のような光沢、散孔材
- **色調**: 白〜淡い黄白色、上品な明るさ
- **硬度**: 硬（気乾密度 0.67g/cm³）
- **分類**: 散孔材・広葉樹
- **lineSpacing**: 2.6
- **lineWeight**: 0.85
- **grainStrength**: 0.5（緻密な木目が繊細な干渉を生む）
- **レーザー加工**: ○ 白い材に焦げ色のコントラストが美しい
- **V溝NC加工**: ◎ 硬いが緻密で美しい光沢の切削面
- **経年変化**: 白色→淡い飴色に穏やかに変化、紫外線でゆっくり黄変
- **文化的意味**: 紅葉の美しさ、岩手の秋の風景、楽器材（バイオリン裏板）としての品格

### 唐松（カラマツ / Larix kaempferi）
- **木目特性**: 明瞭な年輪、脂条、力強い
- **色調**: 赤褐色、脂分で経年変化が独特
- **硬度**: 中〜硬
- **lineSpacing**: 2.3
- **lineWeight**: 1.0
- **grainStrength**: 0.55
- **レーザー加工**: ○ 脂で焦げ具合が変わる（味になる）
- **V溝NC加工**: ◎ 脂があるため工具のメンテナンスに注意
- **文化的意味**: 岩手の構造材、紅葉する針葉樹、変化を恐れない強さ

## 樹種選択ガイドライン

### 初めての方（デモ・ワークショップ向け）
→ **杉**（加工しやすい、失敗しにくい、コスト最安）

### 高級感を求める場合
→ **欅** or **オニグルミ**（V溝NCモード推奨）

### 赤ちゃんの名前に桜がつく場合
→ **ヤマザクラ**（物語性の強化）

### 力強い印象
→ **栗** or **唐松**（力強い木目・環孔材の存在感）

### 優しい印象
→ **ヤマザクラ** or **イタヤカエデ**（優しい色調・緻密な木目）

### 白い材がほしい場合
→ **イタヤカエデ**（最も明るい色調、コントラストが美しい）

### 市産材 vs 県産材
- **盛岡市産**: スギ, カラマツ, アカマツ, ナラ, クリ（地元の山から）
- **岩手県産**: ヤマザクラ, オニグルミ, イタヤカエデ, ケヤキ（県の広葉樹林から）
