# エクスポート検証基準 — Export Validation Criteria

## 検証対象フォーマット

### SVG
- [ ] valid XML（パーサーエラーなし）
- [ ] viewBox が 0 0 350 350（70mm × 5倍）
- [ ] stroke-width が加工線幅（0.1mm以上）に対応
- [ ] テキスト要素がある場合、フォントフォールバック指定あり

### PNG
- [ ] 解像度: 350×350px（標準）または指定サイズ
- [ ] Canvas.toDataURL() 由来のbase64が正常デコード可能
- [ ] 透過背景の場合、アルファチャンネルが正常

### DXF
- [ ] ヘッダーにバージョン情報（AC1015以上）
- [ ] ENTITIES セクションに描画データあり
- [ ] 座標系がmm単位（70×70の範囲内）
- [ ] V溝NCモード用の加工パスとして有効

### デザインストーリーカード
- [ ] パラメータ情報（名前、生年月日、体重等）が画像内に含まれる
- [ ] テキストの可読性（フォントサイズ、コントラスト）
