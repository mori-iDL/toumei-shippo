# スクリーンショット比較 — Screenshot Comparison

## 評価指標

### SSIM（Structural Similarity Index）
- **異なるテストケース間**: SSIM < 0.7 が目標（十分に異なる）
- **同一テストケース再実行**: SSIM = 1.0（完全再現＝決定論的描画の証明）
- **軽微な調整後**: 0.7 < SSIM < 0.95（変化はあるが破壊的でない）

### ピクセル差分
- 差分画像を生成し、変化領域を赤色でハイライト
- 差分率（変化ピクセル数 / 全ピクセル数）を算出

## テストケースペア（標準）

| ペア | 条件A | 条件B | 期待SSIM |
|---|---|---|---|
| 季節差 | 春・はるか | 冬・めい | < 0.5 |
| 名前差 | 春・はるか | 春・りく | < 0.7 |
| 時刻差 | 正午 | 真夜中 | < 0.8 |
| 同一再実行 | 春・はるか(1) | 春・はるか(2) | = 1.0 |
