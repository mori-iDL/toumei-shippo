# ロードマップ整理プラン

## やること

コミット履歴・Notion・コードの現状を横断的に整理し、「どこから来て、今どこにいて、どこへ向かうか」を一枚の地図にする。

## 整理の結果イメージ

### 1. これまでの軌跡（What we've done）

コミット34本を時系列で5つのフェーズに構造化：

**Phase 1: コアエンジン構築**（2/19-2/20, 6コミット）
- 初版リリース → P0（視覚的多様性）→ 64mm描画エリア＋母音ブレンド → 木材パラメータ → ベクタ出力 → 名前バリデーション
- 三層翻訳アーキテクチャ（入力→アルゴリズム→木材）の骨格が完成

**Phase 2: V溝NC開発**（2/20-2/21, 4コミット）
- モバイル対応 → マルチビット（Dual/Triple）→ grooveWidth構造制御 → レイヤー色分け
- V溝NCモードが一通り動く状態に

**Phase 3: 出力品質改善**（2/21-2/22, 3コミット）
- SVG/DXFの加工ソフト互換性を段階的に修正

**Phase 4: 開発基盤・チーム連携**（2/22-2/25, 8コミット）
- Notion MCP統合、ローカル知識DB、4Agent実装、Hookアーキテクチャ
- チーム共有のプロジェクトHub構築

**Phase 5: V溝再設計＋品質確保**（2/25-2/26, 6コミット）
- SVGエクスポート互換性 → V溝60°固定・面幅ベース再設計 → レーザー3層化
- marching squaresストローク化 → 樹種リスト是正＋lineWidth全面修正
- **DRY化＋108パターン自動安全テスト（構造的対策）**
