## 全体像：「ギフトで始めて、プラットフォームに育てる」

赤ちゃんの誕生データを、自然計算アルゴリズムで木の模様に翻訳する。
パーソナライズされた木のパネルを、出産祝いギフトとして販売し、
成長記録プラットフォームへと進化させる3段階のロードマップ。

---

## Phase 1：出産祝いギフト（〜R7上期）

### 商品
- 基本セット ¥10,000（3樹種パネル + ストーリーカード + パッケージ）
- プレミアムセット ¥12,000（5樹種パネル + ストーリーカード + 木箱）

### ターゲット
- 出産祝いを贈る友人・同僚（25-40代女性）
- 祖父母（孫への特別な贈り物）

### 販路
- 新田さんの木育ネットワーク（最速）
- ECサイト（BASE/Shopify、受注生産）

### チーム役割
- 新田紗希：事業オーナー、木育ネットワーク販路、顧客接点
- 阿部さん：パッケージデザイン、ストーリーカードデザイン、美的品質判定
- Zine：パラメトリックデザインシステム開発、レーザー/NC加工、事業設計

### 収益構造
- 原価 ¥2,860〜3,361/セット → 粗利率 66〜71%
- 受注生産：在庫リスクゼロ
- ¥10,000は出産祝いの相場ど真ん中

### なぜPhase 1はギフトか
- 立ち上げが最速。受注生産で在庫リスクなし
- パーソナライズは「贈り物」で最強の差別化
- 花巻オトモとの差別化が最も効く
- 粗利率が高く、チーム全員の報酬を確保しやすい

## Phase 2：木育体験キット追加（R7）

### 商品の拡張
- Phase 1のギフトセット（継続）
- 木育体験キット ¥8,000〜10,000（3樹種パネル + サンプルブロック + 図鑑カード + Webアプリ体験）

### ターゲットの拡大
- 木育に関心のある親（新田さんの既存顧客層）
- 保育園・幼稚園（教材として）
- 企業・自治体（ワークショップ）

### 新しい販路
- ワークショップ展開（コカコーラWS実績の応用、27名参加で¥22万）
- 保育園・自治体への営業（新田さんの得意領域）

### チーム役割の進化
- 阿部さん：図鑑カード・教材デザインも担当
- 新田さん：木育プログラムとの連動設計

### 収益構造
- ギフト（粗利66-71%） + 体験キット（粗利43-69%）の二本柱
- ワークショップで単価以上の収益（1回 ¥15〜25万）

## Phase 3：成長記録プラットフォーム（R8〜）

### 商品の進化
- スターターキット ¥10,000〜12,000（誕生パネル + 成長記録フレーム）
- 成長記録サブスク 年1回 ¥3,000〜5,000（誕生日に新パネル追加）

### ターゲットの深化
- Phase 1の購入者が自然に移行（誕生パネル → 成長記録へ）
- 自治体連携（出生届→特典としてスターターキット）
- 病院・助産院（退院時ギフト）

### サブスクの体験設計
- 誕生日の1ヶ月前にリマインド → 現在の体重・身長入力
- 新しいパネルを生成 → 加工 → 誕生日に届く
- フレームに追加するたびに「年輪が増える」体験
- 3枚→6枚→9枚…と木箱が埋まっていく

### 収益構造
- スターターキット粗利率 53-75%
- サブスク粗利率 54-78%
- LTV（生涯顧客価値）最大 ¥28,000〜42,000（スターター+年1回×6年）
