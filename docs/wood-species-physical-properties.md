## 3. 樹種別詳細プロファイル

### 3.1 スギ（スギ）— 柔・直

**学名**: *Cryptomeria japonica*
**原産地・産地**: 日本固有種。岩手県盛岡市産材
**旬の採材時期**: 冬季（12月〜2月）— 水分が少なく乾燥が容易

#### 物理特性
- **気乾密度**: 0.38-0.42 g/cm³（軽量級）
- **ヤング率**: 6.5-8.5 GPa（軟材の典型値）
- **ブリネル硬度**: 290-330（軟い）
- **収縮率**: 9-12%（小さい＝乾燥による狂いが少ない）
- **含水率**: 生材 70-120%、気乾 15-20%
- **色調**: 淡紅～白色（製材直後）→ 飴色（経年）

#### 木目・組織特性
- **材種**: 針葉樹（軟材）
- **年輪**: **明瞭**。早材・晩材の色の差が大きい
- **導管**: 欠如（針葉樹）。放射組織は通常組織
- **特殊杢**: なし。通常は直線的な年輪
- **板目**: 木目が荒く、力強い表情
- **柾目**: 平行線模様。落ち着きがある

#### 加工特性

**切削性**
- ✓ 良好。刃持ちが長い
- ✓ 逆目が少ない（比較的どの方向からも加工しやすい）
- ✓ 仕上げ面が滑らか

**レーザー加工**
- ◎ **最適**
- 焦げがクリーム色で均一
- 脂分が少ないため煙・ヤニ問題がない
- 細線彫刻に適している
- **推奨パラメータ（xTool P2S CO2）**: Power 25-35%, Speed 800-1200, Pass 1

**NC/V溝加工**
- ◎ **最適**
- 軟質のため加工が容易
- バリ・ケバが少ない
- 刃物磨耗が遅い
- **推奨ビット**: 60°V溝 / 回転数 12000-15000 rpm / 送り速度 1500-2000 mm/min

#### 経年変化
- **色調変化**: 淡紅色 → 飴色（2-5年） → 深い飴色（10年以上）
- **変化速度**: **中速**（紫外線暴露で加速）
- **仕上げ相性**:
  - オイル仕上げ: ★★★ 最適（色が深まり、艶が生まれる）
  - ウレタン仕上げ: ★★ 可（艶は増すが色の深まりが鈍い）
  - 無塗装: ★★★ 可（経年が味わい深い）
- **耐久性**: 無塗装で10-15年の持続性。屋外は不適（腐朽リスク）

#### 岩手県での用途・評価
- 盛岡市産スギは「北材」として知られ、年輪が細かく品質が高い
- 建築用、製箱、造作材として広く使用
- 木工芸品、銘木材としての価値も高い
- **市場価格**: ¥800-1,200/m³（製材材）

#### Zine工房での実装パラメータ
- `lineSpacing: 2.0` — 年輪が細かいため、パターン間隔は広め。繊細さを引き出す
- `lineWeight: 0.6` — 軟材なため細い線でも加工容易。繊細な表現を優先
- `grainStrength: 0.25` — 年輪特性は控えめに。アルゴリズムの多様性を表現

### 3.2 カラマツ（唐松）— 堅・環

**学名**: *Larix kaempferi*
**原産地・産地**: 日本固有種。岩手県盛岡市産材。北海道産も良質
**旬の採材時期**: 冬季（11月〜2月）

#### 物理特性
- **気乾密度**: 0.55-0.60 g/cm³（中程度 → 重い方）
- **ヤング率**: 11.0-13.5 GPa（硬い方。スギの1.5-1.8倍）
- **ブリネル硬度**: 450-550（中～硬）
- **収縮率**: 12-15%（中程度。乾燥時の狂いに注意）
- **含水率**: 生材 100-150%、気乾 12-16%
- **色調**: 黄白色（製材直後）→ 赤褐色（経年）

#### 木目・組織特性
- **材種**: 針葉樹（中～硬材）
- **年輪**: **非常に明瞭**。早材（黄色）と晩材（赤褐色）の対比が強い。環状に見える
- **導管**: 欠如。年輪の環状構造が特徴
- **特殊杢**: 年輪の環が強いため、板目では「環杢」として装飾的価値あり
- **板目**: ダイナミックで力強い
- **柾目**: 放射線状。洗練された印象

#### 加工特性

**切削性**
- ○ 良好。ただしスギより硬い
- △ 晩材が硬いため、逆目の出やすさに注意
- ○ 仕上げ面は良い

**レーザー加工**
- ○ **良好（脂分に注意）**
- 脂分が多いため、焦げ色がやや濃くなる（黄褐～黒褐色）
- ヤニ（脂分の揮発物）が煙として出る。換気が必須
- 細線は脂分で埋まりやすい
- **推奨パラメータ（xTool P2S CO2）**: Power 30-40%, Speed 600-900, Pass 1-2

**NC/V溝加工**
- ◎ **最適**
- 硬さとしなやかさのバランスが良く、加工性が高い
- バリが少ない。刃物磨耗は中程度
- **推奨ビット**: 60°V溝 / 回転数 15000-18000 rpm / 送り速度 1800-2200 mm/min

#### 経年変化
- **色調変化**: 黄白色 → 濃い赤褐色（1-3年） → より深い褐色（5-10年）
- **変化速度**: **速い**（年輪の色が強いため目立つ）
- **仕上げ相性**:
  - オイル仕上げ: ★★★ 最適（赤褐色が深まり、木目が際立つ）
  - ウレタン仕上げ: ★★ 可（艶は増すが、赤褐色の深みが鈍る）
  - 無塗装: ★★★ 可（経年で色が濃くなり、味わい深い）
- **耐久性**: 無塗装で15-20年の持続性（タンニン含有により防虫性がある）

#### 岩手県での用途・評価
- 盛岡市産カラマツは「唐松」として木工品の高級材として評価
- 家具、装飾パネル、楽器用材としても使用
- 年輪の美しさから「カラマツ杢」は装飾的価値が高い
- **市場価格**: ¥1,200-1,800/m³（製材材）

#### Zine工房での実装パラメータ
- `lineSpacing: 0.8` — 環が密で強いため、パターン間隔は小さく。年輪感を活かす
- `lineWeight: 1.1` — 硬材のため太い線でも加工容易。年輪の力強さを表現
- `grainStrength: 0.6` — 年輪特性を強調。環状パターンを顕在化

**学名**: *Pinus densiflora*
**原産地・産地**: 日本各地。岩手県盛岡市産材
**旬の採材時期**: 冬季（12月〜2月）

#### 物理特性
- **気乾密度**: 0.50-0.55 g/cm³（中程度）
- **ヤング率**: 9.5-12.0 GPa（中程度）
- **ブリネル硬度**: 380-450（中）
- **収縮率**: 11-14%
- **含水率**: 生材 90-140%、気乾 13-17%
- **色調**: 淡紅～黄白色（製材直後）→ 黄褐色（経年）

#### 木目・組織特性
- **材種**: 針葉樹（中材）
- **年輪**: 明瞭。スギとカラマツの中間的な年輪幅
- **導管**: 欠如。樹脂道（脂分を含む）が特徴
- **特殊杢**: なし。通常の年輪
- **板目**: 木目が見やすく、温かみがある
- **柾目**: 平行線状

#### 加工特性

**切削性**
- ○ 良好。ただし脂分がある
- △ 逆目が出やすい
- ○ 仕上げ面は良い

**レーザー加工**
- ○ **良好（脂分に注意）**
- カラマツより脂分が少ないが、なお注意が必要
- 焦げ色は黄褐色～褐色で、比較的均一
- 煙・ヤニは中程度
- **推奨パラメータ（xTool P2S CO2）**: Power 28-38%, Speed 700-1000, Pass 1-2

**NC/V溝加工**
- ◎ **最適**
- 加工性はカラマツと同等かやや易い
- バリが少なく、刃物磨耗も中程度

#### 経年変化
- **色調変化**: 淡紅色 → 黄褐色（2-4年） → より深い褐色（5-10年）
- **変化速度**: **中速**
- **仕上げ相性**:
  - オイル仕上げ: ★★★ 最適（温かい黄褐色が深まる）
  - ウレタン仕上げ: ★★ 可
  - 無塗装: ★★★ 可（経年で色が落ち着く）
- **耐久性**: 無塗装で12-18年（タンニン・脂分により防虫性がある）

#### 岩手県での用途・評価
- 広く使用されている汎用材
- 香りが良く、脂分が防虫効果を発揮
- 建築用、造作材として信頼性が高い
- **市場価格**: ¥900-1,400/m³

#### Zine工房での実装パラメータ

### 3.4 ナラ（楢）— 重・緻

**学名**: *Quercus crispula* （ミズナラ。日本産ナラの代表種）
**原産地・産地**: 日本各地。岩手県盛岡市産材。樹齢100年以上の古木
**旬の採材時期**: 冬季（12月〜1月）

#### 物理特性
- **気乾密度**: 0.70-0.75 g/cm³（重量級）
- **ヤング率**: 12.0-15.0 GPa（硬い）
- **ブリネル硬度**: 650-750（非常に硬い）
- **収縮率**: 10-13%
- **含水率**: 生材 50-100%、気乾 12-15%
- **色調**: 白色～薄い褐色（製材直後）→ 深い飴色（経年）

#### 木目・組織特性
- **材種**: 広葉樹・環孔材（硬材）
- **年輪**: 明瞭。環孔材特有の大きな導管が年輪初期に一列に並ぶ
- **導管**: **非常に大きい**（径 0.5-2mm）。年輪初期に集中（環孔材の特徴）
- **特殊杢**: **虎斑杢（とらふ）** — 放射組織が放射状に見える。高級材の証
- **板目**: 虎斑杢が顕著で、装飾的価値が極めて高い
- **柾目**: 通常の柾目（虎斑杢は柾目に垂直に現れる）

#### 加工特性

**切削性**
- ★★★ **困難**。非常に硬い
- 刃物の選択が重要（高硬度合金刃推奨）
- 逆目が非常に出やすい
- 仕上げ面を出すには高い技術が必要
- 切削音が大きく、加工時間が長い

**レーザー加工**
- △ **困難（推奨しない）**
- 密度が高く、レーザーエネルギーの吸収が大きい
- 細線部分では焼け残りのリスク（焼却不十分で黒く焦げる）
- 高出力が必要だが、通常線幅では過度な焦げが生じる
- **制約**: 線幅が太い場合のみ使用。細線パターンは推奨しない
- **推奨パラメータ（xTool P2S CO2）**: Power 45-60%, Speed 400-600, Pass 2-3（ただし危険性あり）

**NC/V溝加工**
- ◎ **最適**
- 硬さと耐久性から、NC加工に適している
- 刃物磨耗が大きい（超硬合金推奨）
- バリが生じることがある。後処理が必要
- **推奨ビット**: 60°超硬ビット / 回転数 18000-20000 rpm / 送り速度 800-1200 mm/min

#### 経年変化
- **色調変化**: 白色～薄褐色 → 深い飴色（3-5年） → さらに深い褐色（10年以上）
- **変化速度**: **遅い**（密度が高く、外部からの影響を受けにくい）
- **仕上げ相性**:
  - オイル仕上げ: ★★★ 最適（深い飴色が生まれ、虎斑杢が際立つ）
  - ウレタン仕上げ: ★★★ 可（保護性が高く、高級感が出る）
  - 無塗装: ★★ 可（経年は遅いが、最終的に美しい）
- **耐久性**: 無塗装で20-30年以上。屋外でも10年以上の持続性。タンニン含有により防虫性・防腐性が高い

#### 岩手県での用途・評価
- ナラは日本の硬木家具・銘木材の最高峰
- ウイスキー樽としても使用（樽香付与）
- 高級家具、装飾パネル、床材、銘木として取引
- **市場価格**: ¥3,000-8,000/m³（製材材。樹齢・品質で大きく変動）

#### Zine工房での実装パラメータ
- `lineSpacing: 0.65` — 年輪が細かく、虎斑杢が強いため、パターン間隔は極めて小さく
- `lineWeight: 1.3` — 硬材のため太い線で存在感を強調
- `grainStrength: 0.85` — 虎斑杢と年輪の表現を最大化

### 3.5 クリ（栗）— 硬・密

**学名**: *Castanea crenata*
**原産地・産地**: 日本各地。岩手県盛岡市産材
**旬の採材時期**: 冬季（11月〜1月）

#### 物理特性
- **気乾密度**: 0.60-0.65 g/cm³（中～重量級）
- **ヤング率**: 10.0-12.5 GPa（硬い）
- **ブリネル硬度**: 500-600（硬）
- **収縮率**: 11-14%
- **含水率**: 生材 80-120%、気乾 12-16%
- **色調**: 淡黄色～薄い褐色（製材直後）→ 濃い褐色（経年）

#### 木目・組織特性
- **材種**: 広葉樹・環孔材（中～硬材）
- **年輪**: 明瞭。環孔材で、大きな導管が環状に並ぶ
- **導管**: 中程度の大きさ（ナラより小さい）
- **特殊杢**: 年輪の環が強く、環杢として装飾的
- **板目**: 環がはっきり見え、力強い表現
- **柾目**: 平行線状（通常の柾目）

#### 加工特性

**切削性**

## 8. 結論と次のステップ

### 8.1 このレポートの位置付け

このドキュメントは、**とうめいしっぽプロジェクトの樹種パラメータを、物理的・加工的実務と結びつける** ための基礎データベースです。

### 8.2 今後の調査・実装の優先項目

1. **レーザー焦げ色の実測データ化**
   - 9樹種 × 複数パターンでサンプル彫刻を実施
   - RGB値を記録し、アプリ内のプレビューに反映

2. **NC加工パラメータの最適化**
   - Shinx 5軸NCルーターでの実績データの蓄積
   - V溝面幅（0.3/0.5/0.8mm）× 樹種での加工時間記録

3. **経年変化のビジュアル記録**
   - 「赤ちゃんのためのパネル」として、経年変化を記録するプロジェクト
   - 1年・5年・10年での写真撮影と色相測定

4. **Zineの美意識との対応付け**
   - アプリ内の`lineSpacing / lineWeight / grainStrength`が、実物の樹種特性にどう対応するか確認
   - 必要に応じ、パラメータの微調整

### 8.3 Zineへの提案

このデータを基に、P4（UI洗練）タスクで以下を実装することを提案します：

1. **樹種カード化**: 各樹種の物理特性と経年変化イメージを視覚化
2. **加工適性の可視化**: レーザー/NC各モードでの「おすすめ度」を表示
3. **親への説明文生成**: 赤ちゃんのパネルが「10年後どう変わるか」を説明する自動テキスト生成
