## 最上流の問い：「これは何か」

商品の名前と定義が、ターゲット・価格・販路・チーム体制のすべてを規定する。
MTGで最初に議論すべきはここ。

| 定義 | 買う人 | 買う理由 | 価格感度 | 競合 |
|------|--------|---------|---------|------|
| **出産祝いギフト** | 友人・同僚・祖父母 | 「他にない」特別感 | 低い（気持ちの値段） | カタログギフト、名入れグッズ |
| **木育おもちゃ** | 親（自分の子に） | 知育・素材体験・安全 | 中程度（日用品の延長） | 花巻オトモ、木製おもちゃ全般 |
| **誕生記念品** | 親・祖父母 | 唯一無二の記録 | 低い（一生モノの値段） | 手形足形、ファーストシューズ |

**仮説**: 「出産祝いギフト」と「誕生記念品」の境界に最大の市場がある。
自分の子にも買うが、**贈る文脈で最も価値が伝わる**。

## 仮説A：出産祝いギフトセット

### コンセプト
「世界にひとつだけの、木の誕生カルテ」

### ターゲット
- **一次**: 出産祝いを贈りたい友人・同僚（25-40代女性が中心）
- **二次**: 祖父母（「孫に特別なものを」）
- **三次**: 親自身（「自分の子にも欲しい」からのリピート・口コミ）

| ストーリーカード 1枚 | ¥100〜200 | 印刷物。デザインの物語を説明 |
| 木箱 or 布袋 | ¥300〜500 | パッケージング |
| **合計原価** | **¥2,860〜3,361** | |

**プレミアムセット（¥12,000）**
| 内容 | 原価 | 備考 |
|------|------|------|
| パネル 5枚（5樹種） | ¥4,100〜4,435 | |

### 販路
- **新田さんの木育ネットワーク**（教室→口コミ→注文）
- **ECサイト**（BASE/Shopify、受注生産）
- **百貨店・セレクトショップ**（出産祝いコーナー、卸掛率60%で粗利が減る）
- **ワークショップ経由**（体験→即注文。コカコーラWS実績の応用）

### 強み
- 粗利率が高い（66-71%）。価格交渉力がある
- 「贈り物」は価格感度が低い。¥10,000は出産祝いの相場ど真ん中
- パーソナライズ（名前・誕生日が入る）は贈り物で最強の差別化要因
- 赤ちゃんの情報さえあれば受注生産可能。在庫リスクなし

### 弱み・リスク
- 「3枚のパネルで¥10,000」の価値を伝えるストーリーテリングが必要
- 贈る側が赤ちゃんの情報（体重・身長）を知らない場合がある
- 納期（受注→加工→発送）が出産祝いのタイミングに間に合うか
- 花巻オトモとの差別化ポイントを明確に言語化する必要がある

### サブスクの可能性
- **成長記録パネル**（1歳・2歳・3歳の誕生日にその時の体重・身長で新パネルを追加）
- 年1回、¥3,000〜5,000。木箱に追加していく成長の記録
- 「3枚→6枚→9枚」と木箱が埋まっていく体験設計

- **EC**（受注生産。誕生日に合わせて配送）

- **病院・助産院**（退院時のギフトとして）

### 強み
- サブスクの継続理由が自然（子どもは毎年成長する）
- LTV（生涯顧客価値）が最も高い（スターター+年1回×6年=¥28,000〜42,000）
- プラットフォームとしてスケールする（赤ちゃん以外にも展開可能）
- 自治体連携が実現すれば安定的な初期顧客を獲得

- プラットフォームの開発・運用コストが追加で発生
- 自治体連携は時間がかかる（政治的プロセス）
- 最初から大きく出すとリソースが分散する
- 「木に刻む」と言いつつデジタルプラットフォームという矛盾感

### サブスクの設計
- 誕生日の1ヶ月前にリマインドメール → 現在の体重・身長を入力
- 新しいパネルを生成 → 加工 → 誕生日に届く
- フレームに追加するたびに「年輪が増える」体験

## Zine（+ Claude）の仮説

**「AでスタートしてCに育てる」が最も現実的なロードマップ**だと考える。

### Phase 1（〜R7上期）: 仮説Aで立ち上げ
- 出産祝いギフトは立ち上げが最速。受注生産で在庫リスクなし
- 粗利率が高く、チーム全員の報酬を確保しやすい

- 花巻オトモとの差別化（パーソナライズ）が最も効く

### Phase 2（R7）: 仮説Bの要素を追加

- 木のサンプルブロックをセットに追加（体験の厚みを増す）

### Phase 3（R8〜）: 仮説Cへの進化
- 成長記録サブスク開始（誕生パネルを買った人が自然に移行）
- プラットフォーム化（出産以外のマイルストーンにも対応）
- 自治体連携の営業開始

## チーム利益配分モデル（たたき台）

### 前提
- 配分対象は**粗利**（売上 − 原価）
- 原価にはZineの製作工賃（デジタルデザイン費 ¥300/枚）を含む
- 以下は議論のたたき台。MTGで合意を目指す

### 案A：工賃固定 + 売上ロイヤリティ型

Zineは製作工賃を原価として確保。新田さん・阿部さんには**売上に対するロイヤリティ**を支払い、Zineは**粗利からロイヤリティ相当額を差し引いた残り**を受け取る。

| 役割 | 配分方法 |
|------|---------|
| 新田さん | 売上の10〜15%（販路・顧客基盤の対価） |
| 阿部さん | 売上の5〜8%（デザイン監修の対価） |
| Zine | 製作工賃（原価内 ¥300/枚）+ 粗利からロイヤリティを引いた残り |

**＜基本セット¥10,000 × 1セット販売時の具体例＞**

| | 新田15%の場合 | 新田10%の場合 |
|---|---|---|
| 売上 | ¥10,000 | ¥10,000 |
| 原価（工賃含む） | ¥3,100 | ¥3,100 |
| 粗利 | ¥6,900 | ¥6,900 |
| 新田さん（売上×%） | ¥1,500 | ¥1,000 |
| 阿部さん（売上×%） | ¥800 | ¥500 |
| Zine（粗利−上記2名分） | ¥4,600 | ¥5,400 |

- **長所**: 新田・阿部さんにとって分かりやすい。売れた分だけ入る
- **短所**: Zineが最もリスクを負う構造。売上が少ない時にZineだけが赤字の可能性

### 案B：粗利配分型（役割別の配分比率）

粗利を役割の大きさに応じて配分。

| 役割 | 配分比率 | 月20個時の月収目安 |
|------|---------|-----------------|
| 新田さん | 25% | ¥34,500 |
| 阿部さん | 15% | ¥20,700 |
| Zine | 50%（= 100% − 25% − 15% − 10%） | ¥69,000 |
| 共通経費 | 10% | ¥13,800 |

**※基本セット¥10,000のみ、月20個販売時の想定**（粗利 ¥6,900/個 × 20個 = ¥138,000）

- **長所**: 公平感がある。チーム全員の利害が一致
- **短所**: 売上が小さいうちは全員の取り分が少ない

### 案C：フェーズ別変動型

初期は固定報酬で安心感を確保し、成長後はロイヤリティ比率を高める。

**Phase 1（立ち上げ期）**: 阿部さんに初期デザイン一括報酬（¥50,000〜100,000）。新田さんは粗利の20%。Zineは工賃+残り。

**Phase 2以降（安定期）**: 新田さん25%、阿部さん15%、Zine残り、共通経費10%。

- **長所**: 阿部さんの初期負担に報いる。成長インセンティブが全員にある
- **短所**: やや複雑。初期一括報酬の原資が必要（補助金活用の検討）

### 三案の比較

| | 案A: 工賃+ロイヤリティ | 案B: 粗利配分 | 案C: フェーズ変動 |
|---|---|---|---|
| 分かりやすさ | ◎ | ○ | △ |
| 公平感 | △ | ◎ | ○ |
| 成長インセンティブ | △ | ○ | ◎ |
| おすすめ | 小規模スタート | 対等なパートナー | 中長期で成長を目指す |

※詳細な収益シミュレーションは別途「収益シミュレーション図解」を参照

## 数字の注意

- 原価は現時点での概算。**加工時間の実測データ（Zineが今週〜来週で取得予定）**で精緻化する
- レーザー加工費はCLAUDE.md記載の¥480/枚（4分×¥120/分）を使用。実測で更新の可能性あり
- パッケージング（木箱・布袋・カード印刷）の原価は暫定値。仕入先確定後に更新
- 送料は含まれていない（販路によって異なるため）
- ロイヤリティ配分は未反映（チーム合意後に利益配分を再計算）
