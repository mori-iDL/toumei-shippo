## 現在の統一Phase定義

### Phase 1 — 完成品即納
- 事業モデル: 対面販売 + EC
- 商品形態: 完成済みセット（4枚組が基準）
- 体験: 「もらう」— 出産祝い・記念品
- 製造: Zine工房でレーザー加工 + 必要に応じてNC加工
- 価格帯: ¥10,000以下（セット）
- **これがMTG⑩で見せる本編**

### Phase 2 — テーラーメイド（4週間納期）
- 事業モデル: 受注生産 + Webアプリ
- 商品形態: 赤ちゃんデータからカスタム生成
- 体験: 「選ぶ」— 親がWebアプリでパラメータを選択
- 製造: 注文→生成→加工→仕上げ→発送
- 価格帯: Phase1より高め（検討中）
- **これもMTG⑩の本編**

### Phase 3 — サブスクリプション（構想段階）
- 事業モデル: 月額配信
- 商品形態: 成長に合わせた新デザイン配信
- 体験: 「つくる」— 子どもの成長に寄り添う
- ⚠ 価格¥1,980/月はClaudeが入れた数字。未決定
- **MTG⑩では別ページに補足として記載**

### Phase 4 — コミュニティ / プラットフォーム（ビジョン）
- 事業モデル: コミュニティ運営
- 商品形態: ユーザー参加型
- 体験: 「育てる」— 家族と共に育つ存在
- **新田さんの事業計画にもユーザーとの交流・情報発信は明記されている**
- **MTG⑩では別ページに補足として記載**
