## 三層翻訳アーキテクチャ

### Layer 1: 入力パラメータ
- 名前（ひらがな）→ 母音分析 → 副アルゴリズム
- 誕生日 → 季節判定 → 主アルゴリズム（春:フィボナッチ、夏:ボロノイ、秋:反応拡散、冬:フラクタル）
- 出生時刻 → ブレンド比率
- 体重 → 力強さ/繊細さ（質的影響）
- 身長 → 伸びやかさ/凝縮（質的影響）
- 天気 → 乱流度

### Layer 2: 自然計算アルゴリズム
主アルゴリズム（4季節） × 副アルゴリズム（5母音:あいうえお） × 各パラメータの質的影響

### Layer 3: 木の固有特性
樹種ごとのlineSpacing / lineWeight / grainStrength設定

## ぽこぽこパレット製品ストーリー（260120打合せ資料）

- **コンセプト**: 違いがわかると、世界が広がる
- **フォーミュラ**: 森 × デザイン × CNC技術 × 発達理解
- **名前の由来**: ぽこぽこ=触感 + poco a poco(伊)=少しずつ + パレット=創造の台

- **安全塗料**: 木製食器用安全塗料を使用
- **遊び卒業後の価値**: コースター / 小物トレイ / 飾りタイル / インテリア

## 新田の設計思想（2/2メール — 核心）

- 「幼くなりすぎない」デザインであること
- 「正解」を与えず、想像や解釈、選ぶ余白が残ること
- 木目など素材の個性を感じられる余白があること
- デザインを「装飾」ではなく「触覚やリズムの翻訳」として活用
- 目指すのは「かわいいおもちゃ」ではなく **「一生ものの道具」**

## SVG/DXF出力の加工互換性

- SVGのopacity属性が加工ソフトで色を変質させる → 純色化で対応済み
- 副アルゴリズムの繊細な線（0.2mm幅）はNC加工には細すぎる → 閾値フィルタで対応済み
- muted colorは人の目には美しいがソフトウェアの色識別には純色が確実
