## 事業概要

- **事業名**: 盛岡市森林づかいイノベーション事業（補助金事業 R6〜R8）
- **プロダクト名**: とうめいしっぽ（旧称: てもり）
- **ワンフレーズ**: 触れて、選んで、育てる——コードが生む形を、森の木が受けとめる
- **コアコンセプト**: 赤ちゃんの誕生データ（名前・誕生日・体重・身長・出生地・天気）を自然計算アルゴリズムで木の模様に翻訳する。「おもちゃ」ではなく「おもちゃが生まれ続けるシステム」

## 事業の原点（2024/7/1 二次募集最終審査会より）

- **スローガン**: 盛岡の森から、次世代の"生きる力"と、森林の価値を高める木のおもちゃを！
- **問題意識**: 日本のおもちゃ観（ひまつぶし・消耗品）vs ドイツ（教育の道具・職人の作品・産業製品）
- **育む力**: 非認知能力（やり抜く力・周りを見る力・コミュニケーション能力）
- **「100年コイン」の起源**: 新田が個人で作ろうとしていた最小・最低コスト形態。本事業でより良い商品づくりに挑戦
- **初期商品案**（構想段階）: 箱庭おもちゃ「はこもり」、ピタゴラスイッチ型仕掛けおもちゃ（法人向けも）、組み立て式ミニシアター
- **コト売り戦略**: 製品＋体験（木工所見学・製作体験）＋森林貢献（植樹券・募金）をセットにした価値提供

## 補助金

- R6〜R8（3年）: 補助金合計 ¥4,980千円
- R7予算 ¥880,000 — 全額執行済、残¥0
- Zine実績: 269.5h → ¥1,100,000受領 → 実質¥4,082/h（適正¥10,000/h）

## 技術基盤

- **Webアプリ**: シングルファイルHTML/CSS/JS（`src/toumei_shippo_designer.html`）
- **三層翻訳アーキテクチャ**: 赤ちゃんデータ → 自然計算アルゴリズム → 木の固有特性
- **二重出力モード**: レーザー加工用SVG/PNG + V溝NC加工用SVG/DXF
- **加工環境**: xTool P2S（レーザー）、Shinx 5軸NCルーター

## ブランド・商品コンセプト（コンセプトシートより）

### ブランドコンセプト
- **TARGET**: 親子
- **INSIGHT**: あそぶことは好きだけど人生に重要性を見出せない
- **CONCEPT**: あそびをからだの一部に
- **IMAGE**: 猫のしっぽ（あそびのツールになって、気持ちを伝えてくれて、いつでもそっと寄り添う）
- **KEY BENEFIT**: あなたを支えるおもちゃに出会う
- **Mission**: 家族の機能不全を断ち切る
- **Vision**: 親子が親友になる

### 商品コンセプト
- **TARGET**: 小学生以下の子をもつ親
- **CONCEPT**: おもちゃをくらしの一部に。親子とそだつ、あそべる道具
- **ミッション**: ちょっとした遊びを毎日に散りばめる
- **KEY BENEFIT**: 特別な用意も時間もいらず、日常の一コマが遊びになる
- **問題意識**: 「今のおもちゃはマジメすぎる！」— 知育偏重への批判
- **派生商品案**: スマホスタンド、ペンスタンド、靴・傘マーカー、箸置き、トレイ、アロマディフューザー、フォトフレーム
