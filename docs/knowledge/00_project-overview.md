## 事業概要

- **事業名**: 盛岡市森林づかいイノベーション事業（補助金事業 R6〜R8）
- **プロダクト名**: とうめいしっぽ（旧称: てもり）
- **ワンフレーズ**: 触れて、選んで、育てる——コードが生む形を、森の木が受けとめる
- **コアコンセプト**: 赤ちゃんの誕生データ（名前・誕生日・体重・身長・出生地・天気）を自然計算アルゴリズムで木の模様に翻訳する。「おもちゃ」ではなく「おもちゃが生まれ続けるシステム」

## プロダクト形態の変遷

さんさコイン → トーテム → トムテム（北欧風小人） → ボロノイ積み木(tumi-ishi型30mm厚) → **フラットパネル70mm角**（現在）

1年半の発散と収束を経て、パラメトリックデザインのフラットパネルに到達。

## 技術基盤

- **Webアプリ**: シングルファイルHTML/CSS/JS（`src/toumei_shippo_designer.html`）
- **三層翻訳アーキテクチャ**: 赤ちゃんデータ → 自然計算アルゴリズム → 木の固有特性
- **二重出力モード**: レーザー加工用SVG/PNG + V溝NC加工用SVG/DXF
- **加工環境**: xTool P2S（レーザー）、Shinx 5軸NCルーター
