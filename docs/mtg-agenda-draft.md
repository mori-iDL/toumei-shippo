## 議題（全体 120分）

### 1. 「これは何か」を決める（25分）

**背景**: 商品の定義が事業全体の方向を決める。出産祝い / おもちゃ / 記念品 で、ターゲットも価格戦略もチャネルも変わる。

**議論のフレーム**:

| | 出産祝いギフト | 木育おもちゃ | 誕生記念品 |
|---|---|---|---|
| 買う人 | 贈る友人・同僚 | 親（自分の子に） | 親・祖父母 |
| 競合 | カタログギフト、名入れ | 花巻オトモ | 手形足形、ファーストシューズ |
| 強み | パーソナライズ | 木育の専門性 | 唯一無二 |

**決めたいこと**:
- Phase 1はどの路線で行くか
- 「贈る人」と「自分用に買う人」、どちらを主ターゲットにするか
- 花巻オトモさんとの差別化ポイントは何か（新田さんの見解を聞きたい）

### 2. セットの中身を決める（25分）

**背景**: 「セット販売で¥10,000〜12,000」は合意済み。中身はこれから。

**選択肢（たたき台からの抜粋）**:

| 案 | 内容 | 原価 | 粗利率 |
|---|------|------|--------|
| 3樹種パネル + ストーリーカード | シンプル | 約¥2,860 | 約71% |
| 5樹種パネル + ストーリーカード + 木箱 | プレミアム | 約¥4,700 | 約61% |
| 3樹種パネル + サンプルブロック + 図鑑カード | 体験型 | 約¥3,060 | 約69% |

※原価は概算。加工時間実測データで更新予定

**決めたいこと**:
- 何樹種入れるか（3? 5? 全9?）
- パネル以外に何を入れるか
- パッケージングの方向性（箱？袋？）

### ☕ 休憩（5分）

### 3. チーム体制と利益配分（30分）

**背景**: MTG⑨で「システム方向への転換」が議論された。阿部さんの役割も転換期にある。加えて、チーム全員の利益配分モデルを具体的に議論したい。

**3-a. 阿部さんの役割（15分）**

提案している新しい役割:
- **Seed Designer**: アルゴリズムの初期パラメータ範囲を設計
- **Aesthetic Gate**: 生成されたパターンの美的品質を判定
- **Story Architect**: 赤ちゃんの物語とデザインの接続を監修
- **加えて、パッケージング・印刷物のデザイン**も自然な領域

決めたいこと:
- 阿部さん自身がどの役割に興味を持つか
- 最初に着手してほしいことは何か

**3-b. 利益配分モデル（15分）**

3つの案をたたき台として用意した（事業構想たたき台に詳細あり）:

| | 案A: 工賃+ロイヤリティ | 案B: 粗利配分 | 案C: フェーズ変動 |
|---|---|---|---|
| 概要 | 売上×%のロイヤリティ | 粗利を役割比率で配分 | 初期は固定、成長後はロイヤリティ |
| 分かりやすさ | ◎ | ○ | △ |
| 公平感 | △ | ◎ | ○ |
| おすすめ | 小規模スタート | 対等なパートナー | 中長期成長 |

決めたいこと:
- どの配分モデルに近い形がいいか（方向性だけでOK）
- 報酬の形（一括 / 継続 / ロイヤリティ）の好み
- 具体的な比率は次回以降でも可。まずは方向感の合意

### 4. 販路と立ち上げスケジュール（20分）

**背景**: R6〜R8の補助金事業の中で、Phase 1をいつ立ち上げるか。

**販路の選択肢**:
1. **新田さんの木育ネットワーク**（最も速い。既存の信頼関係）
2. **EC（BASE/Shopify）**（受注生産。全国対応）
3. **ワークショップ**（企業・自治体向け。高単価。コカコーラ実績あり）
4. **百貨店・セレクトショップ**（出産祝いコーナー。卸は粗利率が下がる）

**決めたいこと**:
- Phase 1はどの販路から始めるか
- 受注から納品までのリードタイム目標
- 月間何個の生産を目標にするか

### 5. サブスクリプションの方向性（10分、時間があれば）

**背景**: Phase 2以降の収益構造としてサブスクの可能性。今回は方向性だけ確認。

**案**:
- 成長記録パネル（年1回、誕生日に新しいパネルを届ける）
- 季節の木育パネル（年4回、季節の樹種で）

**確認したいこと**:
- サブスクをやる意志があるか
- やるとしたらどの形が魅力的か

### バッファ（5分）
- 積み残し事項の確認
- 次回MTGまでの宿題整理
