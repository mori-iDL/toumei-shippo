## 今夜の核心 — 4Phase構想

サブスクモデルの深掘りから生まれた転換:

```
Phase 1: もらう（完成品）→ 受動
Phase 2: 選ぶ（テーラーメイド）→ 半参加
Phase 3: つくる（クリエイティブPF）→ 創造
Phase 4: 育てる（コミュニティ）→ 共創
```

**核心**: サブスクは「物を届ける」ではなく「つくる力へのアクセス権」
- Phase 3アプリ: Playground（スライダー）→ Studio（ノードエディタ）→ Code（0→1開発）
- キャンバス拡張: 70mm → ペンプロッター・家具・3Dプリンタ（SVGは媒体を選ばない）
- Phase 4: Discord/コミュニティ → ハッカソン → 県立大・IPUイノベーションセンターとの産学官連携
